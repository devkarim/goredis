# Goredis

## Overview
Goredis is a clone of [Redis](https://redis.io) - an In-Memory database.

This project is made for educational purposes only, to grasp how low-level projects work behind the scenes.

Inspired by [Build Redis from scratch by Ahmed Ashraf](https://www.build-redis-from-scratch.dev/), [Build Your Own X](https://github.com/codecrafters-io/build-your-own-x)

## Features
- [√] TCP server
- [√] RESP parser
- [√] Command handlers
- [√] Caching layer
- [√] Sharding
